# Web Backend проекта Nofelet

Web бэкенд для сервиса звонков, на Go.

## 🏗 Архитектура

Проект состоит из трех основных компонентов:

### 1. Signaling Server (`/cmd/signaling`)
**Реализовано** ✅
- WebSocket сервер для обмена SDP offer/answer и ICE candidates
- Управление подключениями между клиентами

### 2. SFU (Selective Forwarding Unit) (`/cmd/sfu`)
**В разработке** 🚧
- Медиасервер для маршрутизации аудио/видео потоков
- Поддержка групповых видеоконференций
- Адаптивная битрейт под клиентские устройства

### 3. HTTP Server (`/cmd/web`)
**В разработке** 🚧
- REST API для управления пользователями и комнатами
- Статический хост для Web клиента
- Аутентификация и авторизация

## 🚀 Быстрый старт

### Предварительные требования
- Go 1.23+

### Установка и запуск

```bash
# Клонирование репозитория
git clone git@github.com:MaximTretjakov/nofelet-web.git

# Сборка signaling сервера
make build_web

# Запуск signaling сервера
make run_web

```
